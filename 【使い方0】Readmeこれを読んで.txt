【使い方】（ja）
ここでは、ダウンロードの仕方からサイトを開くまでを説明します。
＊モバイルではダウンロードすることはおすすめしません。

ダウンロード（Googleドライブ）
1.「参考文献作成(フォルダー)」という”フォルダ”を選択（1回クリック）します。
2.ダウンロードします。ダウンロードは画面上部のバーからダウンロード（箱の中に「↓」マーク）（ない場合は3点マークの中）します。
3.ダウンロードが完了したら「ファイル」（もしくは「フォルダー」）を開きます。（ない場合は「ファイル」（エクスプローラー）ソフトから）
4.このままでも使えますが、zipになっていますので、右クリック（二本指でクリック）し、「すべて展開」を押します。
5.展開すると、中に「参考文献作成(フォルダー)」が入っています。それのアイコンをドラックし、左側の「ダウンロード」にドロップします。
(zipは消してOKです)

開く
1.先ほど移した「ファイル」ソフト（もしくは「エクスプローラー」）の「参考文献作成（フォルダー）」をダブルクリックします。
2.そして「index.html」（もしくは「index」）をダブルクリックします。
3.ブラウザで開くことができます。

【How to Use】 (en)

This section explains how to download the files and open the site.
Downloading on mobile devices is not recommended.

Download (Google Drive)

1.Click once to select the folder named “Reference Generator (Folder)”.
2.Download the folder.
Click the Download button in the top bar (the icon with a box and a downward arrow).
If it is not visible, open the three-dot menu and select Download.
3.After the download is complete, open Files (or Folder).
If it does not open automatically, open it from the File Explorer application.
4.You can use it as is, but the file is compressed as a ZIP file.
Right-click (or two-finger click) and select “Extract All”.
5.After extraction, a folder named “Reference Generator (Folder)” will appear inside.
Drag its icon and drop it into the Downloads folder on the left side.
(You may delete the ZIP file after extraction.)

Open

1.Double-click the “Reference Generator (Folder)” in Files (or File Explorer).
2.Double-click “index.html” (or “index”).
3.The site will open in your web browser.
